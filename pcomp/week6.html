  • What language is Processing based on?
    Java
  • In your own words, what is the difference between a global and a local variable?
    A global variable can be used anywhere in its program. A local variable can only be used within the scope of the method/statement/loop where it is declared.
  • What data type does the draw() function return when it is called?
    void - it doesn't return anything
  • In your own words, explain in detail what the code below the "//keep the circle in bounds" comment does
    It detects when the edge of the circle touches the edge of the window, and it reflects its velocity in the y direction. Basically, it reverses the circle.
