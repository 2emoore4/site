drawing update
Gotta keep working or I'm going to go crazy. So I'm working on the drawing renderer.

Let's look through the commit log and see what's new. There's a lot. So last time, I had some 2d shapes and some 3d shapes, but I was kinda faking the 3d shapes. I'm pretty sure I've ported most of my old 3d renderer. I have a proper geometry class which supports cube, sphere, cylinder, and torus primitives. Except they don't really look hand drawn, but whatever. All of the matrix transformations work properly, and geometries can even have children, which transform properly. All of these changes were behind the scenes, so it all looks the same as the last demo.

The next thing I did was make it so that the 2d shapes are actually 3d. They're just fake 2d (all of their z coordinates are 0). This way, I can rotate them and move them all I want. Plus I can even add 2d shapes to 3d geometries, or vice versa, which might be useful.
